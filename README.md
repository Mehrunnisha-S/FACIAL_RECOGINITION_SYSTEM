# Facial Recognition System

## Description

This project recognizes known faces using a webcam.

## Features

- Real-time face detection
- Face recognition
- Unknown face detection
- Webcam support

## Technologies Used

- Python
- OpenCV
- face_recognition
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
